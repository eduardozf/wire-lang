# Changelog

All notable changes to Wire Lang will be documented in this file.

## 0.2.0

- Added bus-rail layout support for clearer shared-net routing.
- Expanded the standard parts catalog and updated rendered documentation examples.

## 0.1.0 - Initial MVP

- Added the `wire-lang` aggregate package with the `wire` CLI.
- Added `@wire-lang/core` with parse, compile, layout, and SVG rendering APIs.
- Added `@wire-lang/cli` with `check`, `render`, and `watch` commands.
- Added the initial `.wire` language surface for components, nets,
  connections, annotations, local component definitions, and render hints.
- Added deterministic SVG output with accessible title/description metadata.
- Added repository docs for the MVP scope, contribution policy, security
  reporting, and architecture decisions.
